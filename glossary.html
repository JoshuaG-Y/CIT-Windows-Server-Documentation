Glossary

Definitions of Key Terms and Technologies Used

Common Terms

Active Directory (AD)
A directory service developed by Microsoft for Windows domain networks. It is included in most Windows Server operating systems as a set of processes and services. Primarily used for centralized domain management, authentication, and authorization.
AppCmd.exe
The primary command-line tool for managing IIS (Internet Information Services) 7 and later. It can be used to configure sites, applications, application pools, and more.
Application Pool (IIS)
A set of one or more worker processes that share the same configuration and application boundaries in IIS. Application pools provide isolation between web applications.
DNS (Domain Name System)
A hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network. It translates human-readable domain names (e.g., www.drhscit.org) to machine-readable IP addresses.
Firewall
A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules. Windows Server includes Windows Defender Firewall with Advanced Security.
Group Policy (GPO)
A feature of Windows that facilitates centralized management and configuration of operating systems, applications, and users' settings in an Active Directory environment. Local Group Policy can be used on standalone servers.
HTTPS (Hypertext Transfer Protocol Secure)
An extension of HTTP used for secure communication over a computer network. HTTPS is encrypted using Transport Layer Security (TLS) or, formerly, Secure Sockets Layer (SSL).
IIS (Internet Information Services)
Microsoft's web server software included with Windows Server. It is used to host websites, web applications, and other web services.
IP Address (Internet Protocol Address)
A numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication. Can be static (fixed) or dynamic (assigned by DHCP).
KeePass
A free and open-source password manager, primarily for Windows. It helps manage passwords in a secure way by storing them in an encrypted database (.kdbx file).
NAT (Network Address Translation)
A method of remapping one IP address space into another by modifying network address information in IP header of packets while they are in transit across a traffic routing device. Often used to allow multiple devices on a private network to share a single public IP address.
NLA (Network Level Authentication)
A security feature in Remote Desktop Services (RDS) that requires a user to authenticate to the RD Session Host server before a full RDP session is established. This helps mitigate denial-of-service attacks.
OS (Operating System)
Software that manages computer hardware and software resources and provides common services for computer programs. In this project, Windows Server 2025.
PAT (Port Address Translation)
An extension of NAT that permits multiple devices on a LAN to be mapped to a single public IP address. The goal of PAT is to conserve IP addresses. It maps multiple private IP addresses to a single public IP address by using different port numbers.
PowerShell
A task automation and configuration management framework from Microsoft, consisting of a command-line shell and associated scripting language built on the .NET Framework.
RDP (Remote Desktop Protocol)
A proprietary protocol developed by Microsoft, which provides a user with a graphical interface to connect to another computer over a network connection.
Robocopy (Robust File Copy)
A command-line directory and/or file replication command for Microsoft Windows. It is known for its ability to reliably copy files and folders, especially over networks, and offers features like mirroring, retry capabilities, and logging.
SQL Server (Microsoft SQL Server)
A relational database management system (RDBMS) developed by Microsoft. It is used to store and retrieve data as requested by other software applications.
SSH (Secure Shell or Secure Socket Shell)
A cryptographic network protocol for operating network services securely over an unsecured network. Typical applications include remote command-line, login, and remote command execution.
SSL/TLS (Secure Sockets Layer / Transport Layer Security)
Cryptographic protocols designed to provide communications security over a computer network. TLS is the successor to SSL. They are widely used for securing web communications (HTTPS) and other network services.
Static IP Address
A fixed IP address that is manually configured for a device on a network, as opposed to a dynamic IP address which is assigned by a DHCP server and can change.
System State
In Windows Server Backup, System State includes critical operating system files, the registry, boot files, Active Directory (if a domain controller), Certificate Services database, IIS metabase, and other components necessary to recover the server's operating system to a functional state.
UAC (User Account Control)
A security feature in Windows that helps prevent unauthorized changes to the operating system. UAC prompts for permission or an administrator password before performing actions that could potentially affect the computer's operation or change settings that affect other users.
VIP (Virtual IP Address)
An IP address that doesn't correspond to an actual physical network interface. VIPs are often used for load balancing or network redundancy. In this project, it refers to the public-facing IP address for drhscit.org managed by the school's firewall.
VPN (Virtual Private Network)
Extends a private network across a public network, enabling users to send and receive data across shared or public networks as if their computing devices were directly connected to the private network.
Windows Server Backup (WSB)
A feature in Windows Server that provides backup and recovery capabilities for the operating system, files, folders, volumes, and applications.
