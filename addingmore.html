Windows Server 2025 Setup Guide

Published on drhscit.org

Requirements

  • Windows Server 2025 installed and running on CIT Hardware
  • Virtual IP behind drhscit.org
  • Accessible via RDP and SSH
  • Proof of concept full stack web application running on IIS and SQL Server (Developer Edition)
  • Install KeePass credential manager
  • Recommendations for any useful software
  • Secured following best practices
  • Documentation published to drhscit.org as a static website/GitHub Pages site (must sync with GitHub)

Setup Instructions

1. Install Windows Server 2025

Ensure that Windows Server 2025 is installed on CIT hardware. Follow the official installation guide provided by Microsoft.

2. Configure Virtual IP

Set up a virtual IP address behind drhscit.org. Use your network configuration tools to assign the IP.

3. Enable RDP and SSH

Enable Remote Desktop Protocol (RDP) and Secure Shell (SSH) access for remote management. Ensure proper firewall rules are in place.

4. Deploy Proof of Concept Application

Deploy a full-stack web application on IIS and SQL Server (Developer Edition). Test the application to ensure it is functional.

5. Install KeePass

Download and install KeePass credential manager to securely store and manage credentials.

6. Recommended Software

  • PowerShell 7
  • Windows Admin Center
  • Visual Studio Code
  • Git for Windows

7. Secure the Server

Follow best practices for securing Windows Server, including enabling Windows Defender, configuring firewalls, and applying security updates.

8. Publish Documentation

Publish this documentation to drhscit.org as a static website using GitHub Pages. Ensure the site syncs with the GitHub repository.
